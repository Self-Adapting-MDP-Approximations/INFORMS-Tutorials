# Weakly Coupled MDP Directory Guide

This directory contains the Python code and notebooks for the weakly coupled
MDP tutorial. The source files implement the model objects and approximation
methods; the notebooks show how to build example instances and compare the
resulting policies.

## Directory Layout

| Path | Description |
| --- | --- |
| `wmdp.py` | Core weakly coupled MDP data structures: component state spaces, linking constraints, feasible joint actions, and model-building helpers. |
| `lagrangian.py` | Baseline Lagrangian relaxation that enforces linking constraints in expectation and produces a repaired feasible policy. |
| `fnr.py` | Feasible network relaxation utilities, including network construction, visualization helpers, FNR optimization, and FNR policy extraction. |
| `delayedallocation.py` | Delayed-allocation/action-generation model that starts from restricted joint actions and refines them by separation. |
| `policy.py` | Shared policy interface and action-vector type aliases used by the simulator and optimization methods. |
| `simulator.py` | Sample-path simulator for evaluating policies generated by the Lagrangian, FNR, and delayed-allocation methods. |
| `helper.py` | Small notebook support utilities. |
| `package_dependencies.yml` | Conda environment specification for the tutorial dependencies. |
| `notebooks/` | Jupyter notebooks for the guided tutorial examples. |


## Notebooks

| Notebook | Purpose |
| --- | --- |
| `notebooks/how-code-works.ipynb` | Guided tour of the codebase. It introduces the weakly coupled MDP objects, builds the dynamic fashion-assortment example, solves the Lagrangian baseline, and explains where the self-adapting methods enter. This is the best starting point for understanding the files in this directory. |
| `notebooks/fnr-self-adaptation.ipynb` | Builds the fashion-assortment WMDP, solves the Lagrangian baseline, constructs the feasible network relaxation, solves the FNR model, and compares the FNR policy with the baseline policy. |
| `notebooks/delayed-allocation-self-adaptation.ipynb` | Builds the same fashion-assortment WMDP and Lagrangian baseline, then solves the delayed-allocation model by starting from a restricted joint-action set and adding useful feasible actions through refinement. |
| `notebooks/text_example.ipynb` | Implements the Section 4.3 illustrative knapsack-bandit example from the tutorial text. It focuses on a small three-product instance used to demonstrate the feasible network relaxation. |

## Suggested Reading Order

1. `notebooks/how-code-works.ipynb`
2. `notebooks/fnr-self-adaptation.ipynb`
3. `notebooks/delayed-allocation-self-adaptation.ipynb`
4. `notebooks/text_example.ipynb`
