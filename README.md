# INFORMS Tutorials

This repository contains the code and tutorial material for **Self-Adapting MDP Approximations**, developed as part of an INFORMS Tutorials project for researchers, PhD students, and advanced readers interested in approximate dynamic programming and adaptive methods for Markov decision processes. It is designed to connect theory and computation through readable implementations, tutorial notebooks, and supporting references so that readers can move from conceptual understanding to reproducible experimentation.

The project is intended as a public-facing scholarly resource rather than a classroom platform: a place to study the methods, inspect the implementation, and engage with the research community through questions, discussion, and future extensions. The repository is maintained by [Andre Cire](https://utsc.utoronto.ca/mgmt/andre-cire), [Selva Nadarajah](https://selvan.people.uic.edu/), [Parshan Pakiman](https://parshanpakiman.github.io/), and [Negar Soheili](https://www.negar-soheili.com/), and is released under the MIT License.
