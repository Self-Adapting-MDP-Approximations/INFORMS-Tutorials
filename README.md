# INFORMS Tutorials

This repository contains the code and tutorial material for **Self-Adapting MDP Approximations**, developed as part of an INFORMS Tutorials project for researchers and readers interested in approximate dynamic programming and adaptive methods for Markov decision processes. It is designed to connect theory and computation through readable implementations, tutorial notebooks, and supporting references so that readers can move from conceptual understanding to reproducible experimentation.

- Research code implementing self-adapting approximation methods, including:
  - Self-adapting network relaxations for weakly coupled Markov decision processes: [https://pubsonline.informs.org/doi/10.1287/mnsc.2022.01108](https://doi.org/10.1287/mnsc.2022.01108)
  - Delayed Allocation in Marginalized Flow Models for Weakly Coupled Markov Decision Processes: http://dx.doi.org/10.2139/ssrn.6127326
  - Constraint-violation learning, an approximate linear programming method: https://doi.org/10.1287/mnsc.2019.3289
  - Self-Guided Approximate Linear Programs: Randomized Multi-Shot Approximation of Discounted Cost Markov Decision Processes: https://doi.org/10.1287/mnsc.2020.00038

The project is intended as a public-facing scholarly resource rather than a classroom platform: a place to study the methods, inspect the implementation, and engage with the research community through questions, discussion, and future extensions.
